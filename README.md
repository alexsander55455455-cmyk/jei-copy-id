# JEI Copy ID

Небольшой клиентский мод для Minecraft 1.12.2, который позволяет копировать ID предметов из JEI и креативного инвентаря в буфер обмена.

## Описание

Когда нужно быстро узнать registry name предмета для конфигов, скриптов или рецептов, не приходится вручную искать его в F3+H или в интернете. Наведи курсор на предмет в JEI или в меню креатива и нажми **Ctrl+C** — ID скопируется автоматически.

## Как пользоваться

1. Установи **Forge 1.12.2** и **JEI**
2. Положи `jei-copy-id-1.0.0.jar` в папку `mods`
3. Наведи курсор на предмет в списке JEI, в окне рецепта или в меню креатива
4. Нажми **Ctrl+C**
5. ID окажется в буфере обмена, в action bar появится сообщение `Copied: ...`

### Примеры

| Предмет | Скопированный ID |
|---|---|
| Алмаз | `minecraft:diamond` |
| Предмет с метаданными | `modid:item_name:5` |
| Жидкость | `modid:fluid_name` |

Если фокус в строке поиска JEI или креатива, **Ctrl+C** работает как обычно и ничего не копирует.

## Требования

- Minecraft **1.12.2**
- **Forge**
- **JEI** (обязательная зависимость)

## Сборка

```bash
./gradlew build
```

Готовый jar: `build/libs/jei-copy-id-1.0.0.jar`
